# LFU кеш
Реализация LFU кеша на C++.

## Сборка программы-драйвера
Собрать можно при помощи CMake:
```
cmake . -B build
cmake --build build
```

## Использование
На стандартный ввод подаются данные в следующем порядке:
1. Размер кеша
2. Число элементов в кеше
3. Сами элементы (целые числа типа `int`)

Пример:
```
$ ./build/lcdriver
2 3 1 1 1
```
